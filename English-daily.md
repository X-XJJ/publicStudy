
# 日常英语
- 单词、词组、词根词缀：见此
- 句子、段落：见活页笔记

- vt. 及物动词，后面直接加名词，无被动形式，表示某种状态，transitive verb。
- vi. 不及物动词，to 名词，有被动形式，表示某种行为，intransitive verb。
- prep. 介词
