# 除sql外的操作
- `;`分号不省略！
- show databases;     查看显示左右数据库
- select database();  查看当前使用的数据库
- show variables  like 'port';  查看数据库使用端口
- show tables 查看当前用户所有表

进入如mysql -u用户名 -p
