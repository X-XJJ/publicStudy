# 一、统计术语
- 基期：作为对比标准的时期，即 base of xxx   的 xxx
  - eg：A 比 B 多100。现期 比 基期 多。A是现期，B是基期
- 现期：当前的时期

- 增长量：现期 比 基期 变化的 具体值。
- 增长率、增幅、增速：以基期为基础的变化率
  - 增长率 r = 增长量/基期量 = (现期量-基期量)/基期量
  - 增长了100% = 增长了1倍
  - eg：A比B增长了280% = A比B增长了2.8倍 = A是B的3.8倍，A = B(1+2.8)
  
- 年均增长量：(现-基)÷年数n
- 年均增长率
  - 现期量 = 基期量×(1+年均增长率)^年数n
  - eg：存钱本金A 年均涨3%，第一年共A(1.3)，第二年A(1.3)(1.3)，第三年A(1.3)(1.3)(1.3)，第n年A(1.3)^n
  - 复利：计算利息的一种方法，利滚利，今年得到的利息在明年可以加入本金，用总值计算明年的利息

- 同比：历史同期。eg：23年8月同比：22年8月，“同期”取单位最大的。23年同比22年。
- 环比：统计周期。eg：23年8月环比：23年7月，“周期”取单位最小的：月。23年环比22年。
  - 统计周期：可以是 年、月、日、季度……等等

- 比重：占百分比。eg：A占B的比重 = A/B
- 倍数：倍数
- 平均数：平均
- 百分数：占百分比

- 百分点：百分数的单位，用于百分比之间的比较和加减。eg：4个百分点 = 4%，去年增速2%，今年比去年增加1个百分点 = 今年增速3% = 今年增速比去年增加50%（但不说这种说法）

- 成数：占几成。eg：三成 = 十分之三，七成五 = 75%
- 翻番：2的次幂。A比B翻一翻 = A是B的2倍 = A比B涨了1倍，翻n翻 = 原来的 2^n 倍

- 顺差：出口 - 进口。eg：顺出去，往外卖的多，挣钱了，日子过得顺啊
- 逆差：进口 - 出口。eg：逆回来

- “增加值”= GDP
  - “增加值”用来描述 国内生产总值 = GDP = 总的增加值
  - GDP = 第一产业增加值 + 第二产业增加值 + 第三产业增加值
  - XX产业增加值 = XX产业GDP ≠增长量
  - 第一产业：农业。第二产业：工业和建筑业。第三产业：其他。

- 恩格尔系数：食品支出 占 个人消费支出总额 的比重。
  - 比例越低越好，生活水平越高。
  - 60%以上：贫穷 - 50%~60%：温饱 - 40%~50%：小康 - 30%~40%：相对富裕 - 20%~30%：富足 - 20%以下：极其富裕

- 基尼系数：衡量贫富差距、居民收入差距的常用指标。
  - ∈(0,1)，越低越好，不平等程度越低。
  - 国际红线 0.4
  - 0.2以下 绝对平均。0.2~0.3 比较平均。0.3~0.4 相对合理。0.4~0.5 差距大。0.5以上 差距非常大。

- 五年计划
  - 第十四个五年规划：2021-2025

# 三、速算技巧
## 估算原则
- 最多保留三位有效数字（上限），按四舍五入规则，具体可看选项的范围差距，再判断保留有效数字误差的影响范围

- 如何确定 保留几位有效数字 来进行计算，四舍五入以5为界
  - 误差值 = 保留 - 原数。eg：23456，保留2位为23000 误差值456，保留3位为23500 误差值44
  - 误差：误差值/原数（百分比）。eg：对应↑误差约2%、0.2%
  - 任何数 保留3位计算，最大误差约0.5%，千分比。（1005保留为1010时误差=5/1000最大，若不四舍五入，1009保留为1000时误差=9/1000=0.9%）
  - 任何数 保留2位计算，最大误差约5%，百分比。（105保留为100时误差最大）

- 判断原则 保留几位有效数字
  - 选项首位均不同，2位——实际上，这个能看出首位是几就行。看出选项里位数开始不同的那位是几就行
  - 选项首位有相同，3位
  - 选项两两之间差距＞10%，2位（就是＞十分之一）
  - 选项两两之间差距＜10%，3位

## （一）截位直除法——根本方法：截取前n位后直接做除法

### 一步直除：只对【分母】【除数】【横线底下、后面那个】进行放缩估算，一般四舍五入保留有效数字，剩下的直接舍弃，对误差范围有数，灵活保留
  - 只做一次除法 A/B = ...，就能得出选项答案
  - 如果分子分母都截位放缩，会增大误差不方便估计误差范围，可能也没有减少计算量

- 实际计算
  - ∵选项中基本只是数字不同、而小数点位置都一样，只需要确定到不同的第一个数字就可以
  - ∴对于除数，在约等于新值后，新值末尾的0都可以扔了，影响无非是小数点位置，不影响计算出的非0数字
  - eg：59260÷(1+7.9%) = 59260÷1.079 ≈59260÷1079 ≈59260÷1080 ≈5920÷108
  - eg：59260÷(1+7.9%) = 59260÷1.079 ≈59260÷1.08 ≈5920÷108
  - ∴基本等于 A/B 只看B前三位直接除得数字就可以了
  - eg：59260÷(1+7.9%) = 59260÷1.079 ≈59260÷108

- 其实感觉 每一步都在小误差内 约等和估算 啊，看离哪个选项最接近，得对估算会产生多少误差有数，不影响选项，练习思维和数感
- 比如 遇到x09估为x10、遇到x01估为x00……的误差就很小
- 熟悉思维和感觉之后，希望可以达到看数字就可以直接估算的境界


### 多步连除：分子分母同时截位，截出来后剩下的直接舍弃，不用管。——即便保留了原来的位数为0，做除法的时候，也是 看两个数前三位相减啊……
- 多个式子做乘除，(A/B)÷(C/D) = ...，(AE/B)÷(C/DF) = ...，等等，可以上下都截位
  - eg：3732/20167 截位为 373/201 ≈1.xx，则不管小数点在哪儿，选1开头的选项即可。
  - eg：3732/20167 约等为 3730/20100 ≈0.1xx，也就 小数点 往后挪几位，没必要费时间，看有数字的位即可

- 先整理，把数量级差不多的挪到一个分式里，再看情况截位估算
  - eg：{7.6% x 123/(1+7.6%)} ÷{32% x 234/(1+32%)} = 123/234 x 7.6%/32% x (1+32%)/(1+7.6%) ≈1/2 x 1/4 x 132/108

- 依然还是：选项差距比例很大的，分子分母同时截位。差距小的，只截下面的分母

- 分母补充法：分母加减到大整数，同时，分子加减上该差值，再乘以整体约等的倍数，来弥补误差


## （二）估算法：估算小技巧
- 叠除估算
  - 分开估算：主要用在有增速的比重对比
    - 式子整理为 A/B ×C/D，A/B是数字整理，C/D是比率整理，估算A/B后，估下C/D是大于or小于1，然后估结果
  - 交叉估算
    - 多式子多数字，看情况运用结合率，先结合哪几个算再结合哪几个算。

- 乘法估算，出现较少
  - 一大一小，同时放大and减小 【相同的比例】。A x B ≈A↑xB↓
    - eg：99x101≈100x100，按比例1%加减，误差1%x1%=万分之一
    - eg：19.776 x 6.58 ≈20 x 6.5 ≈130，按比例1%加减，19.776加到20差了0.224约为20的1%，于是6.58的1%约为0.0658，6.58减去0.0658 ≈6.51 ≈6.5

- 饼状图估算，一般问阴影区占比
  - 图上画个直角，直角是25%，再二分二分……

- (1+a)/(1+b) ≈1 + (a-b) ，b在±20%内一般误差不影响选项
  - 证明：分子加一个减一个，同考研数学思路啊，数学算术都是通的
  - (1+a)/(1+b) = (1+a+b-b)/(1+b) = (1+b)/(1+b) + (a-b)/(1+b) = 1 + (a-b)/(1+b) ≈1 + (a-b)，把 1+b 估算为1，误差为b

  - 1/10= 10 %
  - 1/11 = 9.1 % = 0.091
  - 1/12 = 8.3 % = 0.083
  - 1/13 = 7.7 % = 0.077
  - 1/14 = 7.1 % = 0.071
  - 1/15 = 6.7 % = 0.067

  - 1/17 = 5.9 % = 0.059

  - 1/19 = 5.3 % = 0.053

- 当 r = 1/n 时，对数字敏感，发现给的百分比r 差不多可以估算为特殊分数 1/n 时

- 第一步列式的时候，就不写出百分比了，还是按习惯，直接写成小数形式吧…联想分数更容易…然后再丢小数点去估算

### （四）分数 A、B 比较大小
- 基础：分子↑，分母↓ = 分数↑
- （1）直除首位：直接除，只算出商的首位，或估一下首位比xx大or小
- （2）变化速度：看两个分数的 分子之间的倍数，分母之间的倍数。看AB之间分子的倍数涨的多、还是分母涨的多
  - eg：A = 13/78，B = 27/151，B分子 是 A分子 的2倍多，B分母是A分母的2倍少，B = A x 2+/2- = A x 1+，所以 B＞A
  - 即 算算27/13=2.xx，151/78=1.xx，想象A的分子分母同乘2.xx记为C（数值上A=C），B的分子=C，分母＜C，所以B＞C=A


# 四、重点题型

- 先看选项，范围，能排除的先排除，感觉能猜的先猜

## （一）简单计算、直接查找类
- 加加减减、比大小、不涉及乘除
- 需要：快，准

##（二）增长率 r
- 增长率加减：直接加减百分数，单位是“百分点”
- 上升、下降的描述，可以画图辅助
- 增长率比较：估换为分数比较
- “扩大”几个百分点：数值上说的是绝对值，速率的正负号看具体描述

- 基期量、现期量
  - 基 = 现/(1+r)，截位直除为核心

## （三）间隔增长率 R
- 隔了几月、几年的增长率

如：2020年比2018年的增速R是？

2018 A = A             2018基期量
2019 B = A(1+r1)       2019比上年增长率r1
2020 C = A(1+r1)(1+r2) 2020比上年增长率r2
2018 A = C/(1+R)

R = r1 + r2 + r1×r2
 - R = (C-A)/A = (1+r1)(1+r2) - 1 = r1 + r2 + r1×r2

以此类推
R = (1+r1)(1+r2)(1+r3)...(1+rn) - 1

- 速算技巧
  - r1 x r2 = a% x b% = ab/10000 = ab/100 %
    - 注意：百分数相乘，注意小数点到底在哪里啊！
    - a% x b% = ab÷100%
  - r1 x r2，一个化为分数，一个不变
  - r1、r2＜10% 时，r1 x r2＜1% = 0.01，可忽略


## （四）混合增长率 R

- 理解：类似 几杯不同浓度不同体积的溶液（单个现期量和增速）、在混合后的浓度（混合增速）

- 混合增长率：大小居中，具体值偏向大的增速。（本质应用基期量，但近似用现期量估算方便）
- 混合增速的范围：大小一定在两个单个增速之间。更窄的范围：一边为 两个增速的中间值，另一边为量大的一方的增速

- 典型：上半年收入A、增速 a，下半年收入B、增速b，a＜b，问全年的增速r？
  - 一定有 a＜r＜b，且r偏向 A、B 之间更大的那个
    - 若A＜B，则 (a+b)/2＜r＜b

- 若硬算这类问题，列式为 混合增长量=单个增长量之和
  - (A+B)r/(1+r) = Aa/(1+a) + Bb/(1+b)
- ≈**(A+B)r = Aa + Bb**
  - →Ar + Br = Aa + Bb，合并同类项，再估算需要计算的
  - 估值，将分母都估算为相等值 进行约分，即1+r≈1+a≈1+b
  - →A(r-a) = B(b-r) 
  - →A|r-a| = B|r-b| →**A x r和a的距离 = B x r和b的距离**
  - →A/B = (b-r)/(r-a) →**量与距离成反比：A、B的量越大，r与a、b的距离越小。**

- eg：上半年60亿、增速10%，下半年30亿元、增速16%。则中间值为13%，全年增速范围为10%~13%

- 注意：总收入、年增速、平均增速、累计增速等等，其实本质就是混合的。“其中”字眼后的，就是单个增长率

- 年均增长率（年的范围的首尾，乘除关系，增长量才是加减关系）
  - 现期量 ÷基期量 = (1+年均增长率)^年数n
  - 现期量 = 基期量 x (1+年均增长率)^年数n
  - 年数通常直接减就行，eg：2015~2018，n=3
  - 把简单的选项代入，或选项之间的中间值代入
  - 一定是代入10%、20%的整数值，在4个选项最中间的、容易计算的
  - 进行估算，再排除选项。不会想让你直接开立方之类的……

- 常用
  - 1.1^3 = 1.331，1.1^4 = 1.46+ = 1.4641
  - 1.2^3 = 1.728，1.2^4 = 2+ = 2.0736

- 年均增长量的“年数”，eg：2015~2017年收入的年均增长量
  - 国考和常规考试中：求2个增长量的平均，2015~2016一个，2016~2017一个，年指的是范围
  - 江苏：求3个的平均，2014~2015一个，2015~2016一个，2016~2017一个，年指的是当年的增长量
  - 其他省考事考：有很小可能求3年的，猜一下，看题是否会多给一年（第一年2014年）的具体量，只给2015增速的不算。eg：求2014-2017年的，一定会给2013-2017的数据

- 增速快慢：估算具体量的倍数


## （五）增长量 Δ
- 增长量 = 现期量/(1+r) x r = 现期/(1+1/r)——最常见
  - 增长量 = 现期量 - 基期量
  - 增长量 = 基期量 x r

- 速算技巧
  - 1、特殊分数，用特殊分数 近似替代r
    - r ≈1/n  增长量 ≈现期/(n+1)
    - r ≈-1/n  增长量 ≈现期/(n-1)
  - 2、截位直除

- 增长量的比较 Aa/(1+a) Bb/(1+b)
  - r↑，则 Δ↑
  - 现期↑，则 Δ↑
  - r、现期，都↑：增长量大，A＞B，a＞b，
  - r、现期，一↑一↓：看乘积，Aa、Bb，将 1+a ≈1+b
  - 当a、b差值＞20%时，或，当Aa、Bb比较接近时，列完整式子估算
  - 正r的Δ＞ 负r的Δ
